Контрольные вопросы:
  • Что такое «система контроля версий»?
  • К какому типу VSC относится Git? В чём главное отличие от других типов?
  • В чём заключается преимущество использование Git?
  • Какие команды Git вы знаете?
  • В чём отличие Git и Git Hub?
    Ответы:
  1. Система контроля версий - это программное обеспечение, которое позволяет разработчикам взаимодействовать с версиями рабочих проектов: создавать, перемещать, структурировать и т.д.
  2. Система управления версиями Git работает по принципу распределенной (децентрализованной) системы контроля версий, в которой каждый участник команды имеет полную копию всех данных проекта.
  3. Git позволяет легко создавать резервные копии и отслеживать изменения в коде на протяжении всей истории проекта. Git автоматически фиксирует и публикует изменения, вносимые сотрудниками, что значительно повышает скорость и эффективность работы над проектом.
  4. git init, git add, git config, git commit.
  5. Разница заключается в том, что Git - это сама программа, которая осуществляет контроль версий, а Github - это сайт, где можно хранить репозитории, созданные в Git.
